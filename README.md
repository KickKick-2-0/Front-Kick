## How to build

### Requirements
- git (2.39.0)<br/>
- node (16.15.0)<br/>
- npm (9.6.7)<br/>

### git clone

```shell
$ git clone https://github.com/KickKick-2-0/Front-Kick.git
```

### Build

```
$ cd /
$ cd Front-Kick
$ npm install
$ npm run build
```
### Run

- npm run dev
